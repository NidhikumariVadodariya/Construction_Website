# Construction Website

## Overview

A responsive construction company website developed to showcase company services, project portfolios, and contact information. The website features a modern design, mobile-friendly layout, and professional presentation suitable for construction and infrastructure businesses.

## Live Demo

https://nidhi-vadodariya-construction-website.netlify.app/

## Features

* Responsive design for desktop, tablet, and mobile devices
* Service and project showcase sections
* Professional construction-themed design
* Contact information section
* Clean and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
